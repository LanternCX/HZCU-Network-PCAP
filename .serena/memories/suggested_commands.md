# Suggested Commands

- `git status --short --ignored` - check tracked/untracked/ignored state, including ignored `.docx` sources.
- `git check-ignore -v <path>` - verify ignore rule source.
- `markitdown <input.docx> -o <output.md>` - convert Word documents to Markdown.
- `serena memories check` - validate Serena memory references from the project root.
