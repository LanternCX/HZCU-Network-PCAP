# **题目2：pcap/pcapng数据包的在线分析与展示**

## 一、实验大作业项目目的

* 掌握网络数据包的分析和展示技术。
* 理解网络协议的工作原理和数据包的结构。
* 学习使用Python及其相关库进行网络数据分析和可视化。
* 培养实际动手能力和问题解决能力。

## 二、实验大作业的项目需求

* 实现pcap/pcapng文件上传,存储,下载基本功能
* 数据包分析与展示，**包括但不限于**以下内容：
  + 数据包的基本信息：时间戳、源 IP、目的 IP、源端口、目的端口、协议类型、数据包长度等。
  + 协议分布统计：各类协议的数据包数量及占比。
  + 流量趋势：数据包数量随时间的变化趋势。
  + 其他关键指标：如捕获的数据包总数、捕获时间范围等。
* 作业提交要求：
  + 1）提交实验报告与完整源代码，代码应有详细注释，结构清晰，能正常运行。
  + 2）提交与AI的完整交互记录文档。
  + 3）录制项目讲解视频。

## 三、实验大作业项目技术架构（建议）

* 前端：Vue3.js、Element Plus
* 后端：python Flask，使用python的各种库提取和分析pcapng数据包的各种信息
